# Event Management System

A simple event management system built with C# and Windows Forms.

## Features
- Login system with username/password authentication
- Dashboard after successful login
- Logout functionality

## Login Credentials
- **Username:** `admin`
- **Password:** `password123`

## How to Run
1. Open `EventManagementSystem.sln` in Visual Studio
2. Press **F5** to build and run
3. Login with the credentials above

## Project Structure
- `Form1.cs` - Login form
- `DashboardForm.cs` - Dashboard form  
- `Program.cs` - Application entry point
- `App.config` - Configuration file

## Technologies
- C# .NET Framework
- Windows Forms
- Visual Studio
